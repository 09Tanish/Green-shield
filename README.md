# Green-shield
AI-powered smart agriculture platform that detects plant diseases, recommends suitable crops based on soil type, provides weather tracking, farming calendar guidance and integrates a blockchain-based token reward system with an e-commerce platform for farmers.


A production-grade AI-powered smart agriculture platform designed to assist farmers in crop monitoring, plant disease detection, and smart farming decisions. This system integrates machine learning, weather tracking, soil analysis, and a blockchain-based token reward mechanism to improve productivity and reduce crop loss.
